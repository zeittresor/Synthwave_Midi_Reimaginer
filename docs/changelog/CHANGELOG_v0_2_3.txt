Synthwave MIDI Reimaginer GUI v0.2.3

- Added BPM slider in GUI; BPM is passed to the engine and stored in analysis output.
- Added Repeated note amount slider to reduce or preserve long same-note loops.
- Added optional Use style lead/melody instruments checkbox; OFF by default.
- Added 22+ new hybrid/non-electronic style presets, 57 total styles.
- Added extra drum-feel render branches for rock, metal, punk, pop, ska, reggae, dub, folk, bossa, latin, jazz and orchestral styles.
- Updated CLI with --bpm, --repetition and --use-style-instruments.
- Analysis TXT now records BPM, repetition and style-instrument setting.
- Random seed remains enabled by default in the GUI.
